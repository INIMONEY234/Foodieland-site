# Foodieland 🍔

A modern, fully responsive food recipe and blog website built with **HTML**, **CSS**, and **JavaScript**.

Designed with a clean aesthetic, smooth mobile navigation, and engaging sections like Hero, Categories, Recipe Grids, Chef Spotlight, Instagram feed, and Newsletter.

---

## ✨ Features

- **Fully Responsive Design** — Looks great on desktop, tablet, and mobile
- **Sticky Header** with smooth hamburger menu + overlay
- **Modern Hero Section** with eye-catching layout
- **Recipe Grids** (Featured recipes + Delicious bottom section)
- **Categories Section**
- **Chef Spotlight**
- **Instagram-style Gallery**
- **Newsletter Signup**
- **Blog Post Page** (`blogpost.html`)
- **Contact Page** (`contact.html`)
- **Custom CSS Variables** for easy theming
- **Smooth animations** and hover effects

---

## 🛠️ Technologies Used

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, Grid, Custom Properties, Media Queries
- **Vanilla JavaScript** — Mobile menu toggle, overlay, and interactions
- No frameworks (pure frontend)
